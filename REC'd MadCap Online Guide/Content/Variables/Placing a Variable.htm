Placing a Variable

 

  1. Open a new or existing topic in the XML Editor.

  2. Select where you want the variable to start.
  3. Click the Insert a variable button in the local toolbar. The Variables dialog box opens.

 

 

  1. Select a Variable Set and Variable.
  2. Click OK.
    • Note: Both the variable name and definition appear in brackets, but only the definition will get transferred to the output.
