Creating a Variable Set

 

  1. Select the Project tab in the Ribbon.
  2. Click the New dropdown menu located in the Content section.
  3. Select Add Variable Set. The Add File dialog box opens.

 

 

 

  1. Make sure New from Template is selected as well as the file called MyVariables.
  2. Select the File Name text field. Enter a name.
  3. Click the Add button. The new variable set appears in the Editor window.
