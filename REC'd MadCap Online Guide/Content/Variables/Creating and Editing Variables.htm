Creating and Editing Variables

 

  1. Open the Project Organizer.
  2. Expand the Variables folder.
  3. Double click the desired variable set. A chart with the variable names, definitions, and comments appears organized in alphabetical order.

 

 

  1. Click the Name cell of a variable and name it.
  2. Select the Definition cell of the same variable and type what you want to appear in the document.
  3. To add a new variable click the Add variable button located in the local toolbar.
  4. Save the file.

NOTE: You can add a comment to clarify the use or describe the variable in more detail. This is useful when multiple people are working on a document or to help differentiate similar variables on a large document.
