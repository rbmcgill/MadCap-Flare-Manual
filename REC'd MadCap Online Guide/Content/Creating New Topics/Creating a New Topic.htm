First we’re going to tackle the process of creating a topic.

 

  1. Start a new project or open an existing project file.
  2. Open the Content Explorer window.
  3. Expand the topic folder you want to place the topic in.
  • Callout: Double clicking the name will have the same effect.
  1. Click the Create a new Topics file button (pic) to create your own topic. The Add File window appears.
    • Note: If the window is too small, expand the window until you see the button in the local toolbar.
    • Note: you can also right-click folder and select New > Topic
    • Note: You can also pres Ctrl + T to make one
  2. Choose the template you wish to base your topic on.
  3. Make sure the Topic Folder is the one you want to nest your topic in.
  4. Name your new topic. If you know what you want your heading within the topic to be, type it under 1st Heading.
    • Callout: The title is the title of the website if you're making a web page.
  5. Apply stylesheets and styles you wish to use.
  6. Click the Add button. Your topic appears in the Editor window.
