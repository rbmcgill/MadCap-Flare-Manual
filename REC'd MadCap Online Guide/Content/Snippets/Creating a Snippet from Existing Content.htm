Creating a Snippet from Existing Content

It is much easier to create a snippet from content that already exists. Once content has been turned into a snippet you cannot edit the content in the Topic file, only in the snippet file.

 

  1. Open a new or existing topic in the XML Editor.

  2. Highlight the content you want as a snippet.
  3. Click the Home tab in the Ribbon.
  4. Select theCreate Snippet button in the Snippet section. The Create Snippet dialog box opens.

 

  1. Name the snippet.
  2. Make sure the snippet project folder is selected in the Resources folder.
  3. Recheck the content of the snippet in the text box.
    • Note: To make a correction, exit out of the Create Snippet box and edit.
  4. Make sure the Replace Source Content box is checked so that if the snippet is edited, this content will change as well.
  5. Click Create.
    • Note: Depending on the viewing options selected, the name of the Snippet along with the actual content appears in the file, but only the content will show up in the output.

 

 

WARNING: Be careful saving a snippet this way because it will be saved in a local file, not the Snippet folder. Creating a Snippet with New Content. Don't Do It.
