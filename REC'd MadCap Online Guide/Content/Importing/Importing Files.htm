Importing Files

 

Working within MadCap Flare, you may need to import different kinds of documents, for example Legacy Docs, into your project.

 

  1. Start a new project or open an existing project folder.
  2. Click the Project tab in the Ribbon.
  3. Click the Import button in the Content section. A dropdown menu appears with all the options for documents and projects you can import.
  4. Select the type of file you wish to import. The Import [File Type Name] Wizard window appears.
  5. Click the Next button if you wish to import it to this project.
  6. Click the Add Files button () to add the file to a list of items you wish to import. The Open File window appears.
  7. Select the file(s) and click the Open button. The files appear in list view in the Import Window.
  1. Callout: Select the doc and click the Open Selected button to make sure this is the file you want.
  1. Click the Next button. The source document is scanned. The Styles Converter dialog box appears.
    • Note: If you want to know more about the extra options for importing styles, go to (link Importing Options).
  2. Click the Finish button when you are ready to import the file. The Accept Imported Documents dialog box appears.
  3. Click the Accept button to import the file(s). Click the Cancel button to go back to the Wizard and make more changes. The file(s) appear as a topic folder within your Content Explorer window.

 

You have now imported a document into your project!
