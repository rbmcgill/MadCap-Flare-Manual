Creating a Condition Tag

 

  1. Open the Project Organizer.
  2. Double click on the Conditional Text folder. The folder opens and displays your existing condition tag sets.
  3. Double click on your condition tag set. The Condition Tag Set Editor opens in the Editor window . Flare provides two default condition tags: a PrintOnly tag for print outputs, and a ScreenOnly tag for online outputs.
  4. Click the New Item button in the local toolbar of the Condition Tag Set Editor.

 

 

Condition tags will not be reflected in your output unless you apply them to your content.
