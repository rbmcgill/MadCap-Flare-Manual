Applying a Condition Tag to XHTML Tags

  1. Open the intended topic. Make sure you are in the XML Editor.
  2. Highlight the desired text you want to apply the condition tag to.
  3. Open the Project Organizer with your text still selected.
  4. Expand the Conditional Text folder.
  5. Expand the conditional text set.
  6. Drag the condition tagto the Tag Bar to the left of the selected text in the XML Editor.
  7. Release the cursor to apply the condition tag.
