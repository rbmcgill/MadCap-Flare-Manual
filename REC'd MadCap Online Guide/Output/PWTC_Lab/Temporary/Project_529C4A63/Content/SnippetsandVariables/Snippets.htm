Snippets

 

Snippets are groups of text, images, or tables that are reused often throughout a document. Rather than retyping the content, a snippet allows you to create it once and add it to multiple topics. When you edit a Snippet, Flare automatically edits all the topics where that Snippet has been placed.

 

 

Creating a Snippet from Existing Content

It is much easier to create a Snippet from content that already exists rather than generating a new Snippet and new content at the same time.

NOTE: Once content has been turned into a snippet you cannot edit the content in the Topic file, only in the Snippet file.

 

  1. In a topic, highlight the content you want as a Snippet.
  2. Click the Home tab.
  3. Select Create Snippet located in the middle of the ribbon. An dialog box opens.

 

  1. Name the Snippet.
  2. Make sure the Snippet project folder is selected in the Resources folder.
  3. Recheck the content of the Snippet in the text box. To make a correction, exit out of the Create Snippet box and edit.
  4. Make sure the Replace Source Content box is checked so that if the Snippet is edited, this content will change as well.
  5. Click Create. Depending on the viewing options selected, the name of the Snippet along with the actual content appears in the file, but only the content will show up in the output.

Placing a Snippet

 

  1. Open the desired topic file.
  2. Place your cursor and select where you want the Snippet to start.
  3. Click Insert a snippet from the toolbar at the top of the window. The Snippet appears.

 

    


 

Editing a Snippet

When you edit a snippet file, any topic that contains that snippet will change as well.

 

  1. Open the Content Explorer.
  2. Open the Resources folder.
  3. Open the Snippets folder.

 

 

  1. Double click the Snippet file you want to edit. The file opens as a window in the workspace.
  2. Edit and save the Snippet.

Variables

Variables are small phrases or words that are used repeatedly throughout a document. This can include phone numbers, addresses, product names, company names, and authors. In order to have a Variable, you need to create a Variable Set which is essentially a folder to keep them in. Flare templates come with an existing variable set called General which includes several commonly used variables.

Creating a Variable Set

 

  1. Select the Project tab in the Ribbon.
  2. Click the New paper icon located in the Content section. A dialog box opens.

 

 

 

  1. Select Variable Set from the dropdown menu.
  2. Make sure New from Template is selected as well as the file called MyVariables.
  3. Name the File.
  4. Click Add. The new Variable Set appears in the workspace.

 

Creating and Editing Variables

 

  1. Open the Project Organizer.
  2. Open the Variables folder and select the desired Variable Set. A chart with the variable names, definitions and comments appears organized in alphabetical order.

 

 

  1. Click the Name section of a variable and name it. This is how you will identify the meaning of each variable.
  2. Select the Definition section of the same variable and type the phrase you want to appear in the document.
  3. To add a new variable click the Add variable icon located in the toolbar.
  4. Save the file.

NOTE: you can add a comment to clarify the use or describe the variable in more detail. This is useful when multiple people are working on a document or to help differentiate similar variables on a large document.

 

 

Placing a Variable

 

  1. Open the desired topic file.
  2. Place your cursor and select where you want the variable to start.
  3. Click Insert a variable. A dialog box opens.

 

 

  1. Select a Variable Set and Variable.
  2. Click OK. Both the variable name and definition will appear in brackets, but only the defnition will get transferred to the output.
