Inserting Images (and Videos? Media?)

 

 

  1. Start a new project or open an existing project folder.
  2. Click the XML Editor tab at the bottom of the Editor window.
  3. Click the Insert tab on the Ribbon. The Multimedia section is the first from the left.
  4. Select a space in the document where you wish to place your image.
  5. Click the Image button in the Multimedia section or within the XML Editor window. The Insert Image window appears.
  6. Select the location of your image.
  7. Click the Open button.
  8. Add a Screen Tip and Alternate Text here for accessibility.
  9. Click the OK button when you are ready to insert the image. The Copy to Project window appears.
  10. Click the OK button in this window to confirm you wish to place the image. The image appears within your topic.

 

Inserting Multimedia is a very similar process except that you can also pull material from the internet.

  1. Start a new project or open an existing project folder.
  2. Click the XML Editor tab at the bottom of the Editor window.
  3. Click the Insert tab on the Ribbon. The Multimedia section is the first from the left.
  4. Select a space in the document where you wish to place your image.
  5. Click the Multimedia button in the Multimedia section. The Insert Multimedia window appears.
  6. Select the location of your video or type in the URL.
  7. Click the Open button. The video appears within your topic.

 

Glossary terms:

A screen tip is a phrase that appears when the end user hovers over the image.

Alternate text displays when the image is not available, such as when a disabled individual is using a screen reader.

http://webhelp.madcapsoftware.com/capture7/Content/Steps/Setting_Screen_Tips_and_Alternate_Text.htm

The tab index is a field that lets you specify the order in which users can tab through images in a topic.
