Creating and Using Topics

See Chapter _____ for more information about the Content Explorer.

 

First we’re going to tackle the process of creating a topic.

 

  1. Start a new project or open an existing project folder.
  2. Open the Content Explorer window.
  3. Click the dropdown arrow to the left of your intended topic folder. The topics within the folder appear below.
  1. Callout: Double clicking the name will have the same effect.
  1. Double click the topic you wish to edit.
  2. Click the Create a new Topics file button (pic) to create your own topic. The Add File window appears.
  3. Choose the template you wish to base your topic on.
  4. Pick the Topic Folder you want to nest your topic in.
  5. Name your new topic. If you know what you want your header within the topic to be, type it under 1st Heading.
  6. Apply stylesheets and styles you wish to use.
  7. Click the Add button. Your topic appears in the Editor window.

 

You have created a topic! Now let me show you how to use it.

 

  1. Click the XML Editor tab at the bottom of the Editor window.
  1. Note: For more info on the Editor window/XML editor/HTML editor, go to _____
  1. Type in the material you would like to use. Tags appear along the left edge of the window.
  2. Click the HTML Editor tab at the bottom of the Editor window.
  3. Edit the tags using XHTML as necessary.

 

You now know how to write and edit text in a topic!
