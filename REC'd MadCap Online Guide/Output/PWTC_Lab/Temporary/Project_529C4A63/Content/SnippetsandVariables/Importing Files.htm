Importing Files

 

Working within MadCap Flare, you may need to import different kinds of documents, for example Legacy Docs, into your project.

 

  1. Start a new project or open an existing project folder.
  2. Click the Project tab in the Ribbon.
  3. Click the Import button in the Content section. A dropdown menu appears with all the options for documents and projects you can import.
  4. Select the type of file you wish to import. The Import [File Type Name] Wizard window appears.
  5. Click the Next > button if you wish to import it to this project. Skip the following step.
  6. Specify that you will Import into a new project to do so. Fill in the name, file location, and output type before clicking the Next > button.
  7. Click the Add Files button () to add the file to a list of items you wish to import. The Open File window appears.
  8. Select the file(s) and click the Open button. The files appear in list view in the Import Window.
  1. Callout: Select the doc and click the Open Selected button to make sure this is the file you want.
  1. Click the Next > button. The source document is scanned. The Styles Converter window appears. All steps after this point are optional.
  2. Click the Finish button when you are ready to import the file.
  1. Note: You can click the Finish button at any point when you are satisfied with the import settings/options. Skip to step 28.
  1. Select the styles you wish to keep and click the right-pointing arrow.
  2. Click the Next > button. The Options… window appears.
  3. Select the options for converting topics, titles, and tables you wish to apply to your file upon importing.
  4. Click the Next > button. The Stylesheet… window appears.
  5. Click the Stylesheet… button to import an existing stylesheet to apply to the file upon importing. The Open window appears.
  6. Select the stylesheet you wish to apply.
  7. Click the Open button. The file location appears in the Stylesheet bar.
  8. Select your preference between preserving and ignoring your file’s existing styles.
  9. Click the Next > button. The Paragraph Style Mapping… window appears.
  10. Select the File Style you wish you change.
  11. Select the Flare Style you wish to map into that File Style.
  12. Click the Map button. The Flare Style will use the File Style upon importing the file.
  13. Click the Next > button. The Character Style Mapping… window appears.
  14. Select the File Style you wish you change.
  15. Select the Flare Style you wish to map into that File Style.
  16. Click the Map button. The Flare Style will use the File Style upon importing the file.
  17. Click the Finish button when you are ready to import the file. The Accept Imported Documents window appears.
  18. Click the Accept button to import the file(s). Click the Cancel button to go back to the Wizard and make more changes. The file(s) appear as a topic folder within your Content Explorer window.
